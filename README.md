Travel Planner - Платформа для планування подорожей

Travel Planner - це веб-платформа для планування та організації подорожей. Користувачі можуть створювати маршрути, управляти бюджетом та зберігати інформацію про свої поїздки.

Мета проєкту

Створення зручного інструменту для:
- Планування подорожей з датами та напрямками
- Контролю бюджету
- Збереження інформації про всі поїздки в одному місці
 
Реалізований функціонал (MVP)

- Реєстрація та авторизація користувачів
- Особистий кабінет з усіма поїздками
- Створення нових подорожей (назва, напрямок, дати, бюджет)
- Перегляд деталей кожної поїздки
- Безпечне збереження даних у базі
